# Orange

*Orange* is a cozy 2D isometric pixel-art puzzle adventure built in Godot 4.x, targeting responsive web as its first platform. The protagonist, Orange — a curly orange-haired girl — explores a rain-touched forest where mushrooms, animal tracks, rhythmic raindrops, and bark symbols form the language of the puzzles; collecting illustration fragments piece by piece reveals a Secret Map Card that bridges her journey toward a sunlit waterfall in the world beyond.
