# Orange Project Brief

## Identity

**Orange** is a cozy isometric pixel-art puzzle adventure about a curly orange-haired girl named Orange.

The world uses mixed nature data: real nature observations are stylized into poetic puzzle rules. Rain, sunlight, mushrooms, animal tracks, bark, water, wind, roots, and mist become the language of the game.

## First Prototype Scope

Build only:

- Episode 1: Rain-Found Forest.
- Episode 2 bridge: Sunlit Map Activation.

Do not build the full waterfall episode yet. The prototype ends when Orange activates the Secret Map Card under sunlight and the waterfall route appears.

## Core Loop

Explore -> notice nature data -> solve puzzle -> collect illustration fragment -> complete key item -> use key item in the next episode.

## Episode 1

- Place: Forest.
- Phenomenon: Rain.
- Emotion: Curiosity.
- Key item reward: Secret Map Card.

Episode 1 puzzles:

1. Mushroom Path Puzzle -> map fragment 1.
2. Animal Tracks Puzzle -> map fragment 2.
3. Raindrop Rhythm Puzzle -> map fragment 3.
4. Wet Bark Symbol Puzzle -> map fragment 4.

## Episode 2 Bridge

- Place: Sunlit clearing near the waterfall route.
- Phenomenon: Bright sunlight.
- Emotion: Hope.
- Required previous key item: Secret Map Card.

Bridge puzzle:

- Sunlight activates the Secret Map Card.
- The player rotates or aligns the map in sunlight.
- The waterfall route appears.
- Prototype ends gracefully.

## Story Continuity Rule

Episode rewards must matter later.

Rain reveals the Map -> Sunlight activates the Map -> Compass follows the River -> Ancient Tree opens the Door.

For the first prototype, only prove the first bridge:

Secret Map Card completed in Episode 1 -> Secret Map Card used in Episode 2 bridge.
